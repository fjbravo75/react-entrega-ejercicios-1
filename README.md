# Entrega React - Ejercicios 1

Primera entrega del módulo de React. El objetivo es rehacer los nueve ejercicios del módulo de JavaScript usando React, componentes funcionales, props, `useState` y `useEffect`.

## Estructura

- `src`: código fuente de la aplicación.
- `src/components`: componentes comunes.
- `src/ejercicios`: un componente por cada ejercicio.
- `ejercicio-1.html` a `ejercicio-9.html`: una página HTML para cada ejercicio.
- `docs`: versión generada para publicar en GitHub Pages.

## Ejercicios

1. Cambiador de color de fondo.
2. Contador de clics.
3. Lista dinámica.
4. Filtro de búsqueda.
5. Calculadora sencilla.
6. Temporizador.
7. Generador de contraseñas.
8. Contador de palabras y caracteres.
9. Lista de tareas con `localStorage`.

## Ejecutar en local

```bash
npm install
npm run dev
```
